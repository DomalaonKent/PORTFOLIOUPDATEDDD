Alphatera Construction Supply

Figma B2B Platform Web App UI/UX

A B2B partner portal UI design for Alphatera Construction Supply. Designed with a navy and rust color system to reflect industrial authority, enabling partners to manage orders, view product catalogs, and track deliveries.

Gallery

Overview

The Alphatera Construction Supply portal is a B2B web platform designed for construction material partners and clients. The design system is anchored by a navy (#1B2A4A) and rust (#C0420A) palette that communicates professionalism and industrial strength.

The goal was to create a partner-facing interface that simplifies order management and product discovery while maintaining a corporate visual identity aligned with Alphatera's brand.

Key Features

  • Partner login and authenticated dashboard with company-level data
  • Product catalog with category filtering and search
  • Order management interface with status tracking
  • About page with leadership profiles and image carousels
  • CoreUI-inspired collapsible sidebar navigation
  • Responsive layouts adapted for desktop and tablet

Design Process

01

Brand Discovery

Studied Alphatera's branding materials and business context. Defined the navy/rust color system to reflect industrial authority and trust.

02

Information Architecture

Mapped out the partner portal's user flows — login, dashboard, catalog browsing, order placement, and account settings.

03

Component Library

Built reusable Figma components: sidebar items, product cards, order status badges, form inputs, and data tables.

04

High-Fidelity Design

Assembled full screens using the component library. Applied typography scale and spacing tokens for visual consistency.

05

Prototype & Review

Created an interactive prototype linking key flows. Reviewed with stakeholders and refined based on feedback.
