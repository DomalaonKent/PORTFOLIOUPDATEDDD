Barbershop App

FigmaMobile AppiOS/AndroidUI/UX

A mobile app UI for a barbershop booking and service management experience. Designed with a dark, masculine aesthetic — combining bold typography, high-contrast imagery, and a streamlined appointment booking flow.

Gallery

Overview

The Barbershop App is a mobile UI concept for a grooming and appointment booking application. The design embraces a premium dark aesthetic with warm amber accents, communicating the brand's positioning as a modern barbershop for style-conscious clients.

The app is designed for a seamless one-handed mobile experience — from discovering services, selecting a barber, picking a time slot, and confirming a booking in as few taps as possible.

Key Features

  • Onboarding flow with service category selection
  • Barber profile cards with ratings and available slots
  • Calendar-based appointment picker with time slot grid
  • Service menu with pricing and duration details
  • Booking confirmation screen with QR code and summary
  • User profile with booking history and loyalty points

Design Process

01

Mood & Direction

Explored visual references for premium grooming brands. Settled on a dark charcoal palette with amber accents, bold serif headings, and editorial-style photography framing.

02

User Flow Mapping

Mapped the core booking flow from app launch to confirmation. Minimized the number of steps and decisions required from the user.

03

Mobile-First Design

Designed all screens at 390×844 (iPhone 14 Pro) with bottom-heavy navigation to support thumb reachability.

04

Component Library

Built reusable mobile components: tab bars, bottom sheets, date pickers, barber cards, and booking confirmation templates.

05

Interactive Prototype

Linked screens into a complete Figma prototype covering the full booking journey with micro-interaction cues.
