Alphatera Construction Supply

Angular CoreUI FastAPI Web Platform

A B2B web platform and admin dashboard for Alphatera Construction Supply. The platform provides an online presence and streamlined management for construction material supply — featuring a public-facing website alongside a CoreUI-powered admin panel for managing services, projects, and team info.

Gallery

Overview

Alphatera Construction Supply needed a modern digital presence that could both market their services and help internal staff manage operations. I was part of the team that designed and developed the full platform — from the public website to the backend admin dashboard.

The public site features a "Building Smart, Supplying Fast" hero section showcasing services, project portfolio, and team. The admin dashboard uses CoreUI components for clean data management.

Key Features

  • Public B2B website with services, projects, and team pages
  • CoreUI-based admin dashboard for content management
  • Services management — add, edit, and display construction services
  • Project portfolio management with image uploads
  • Team directory with role-based profiles
  • FastAPI backend with secure endpoints and database integration

Development Process

01

Design & Branding

Designed the visual identity and UI in Figma. Used a dark, industrial aesthetic to match the construction brand — bold typography with orange/amber accents.

02

Public Website (Angular + Tailwind)

Built the customer-facing site using Angular with Tailwind CSS for responsive styling. Implemented smooth animations and image-heavy layouts.

03

Admin Dashboard (CoreUI)

Integrated CoreUI component library for the admin panel. Built CRUD interfaces for services, team members, and project listings.

04

Backend Integration (FastAPI)

Connected the frontend to a Python FastAPI backend. Implemented file upload, authentication middleware, and RESTful CRUD endpoints.
