R5 V2 Intranet

Figma Enterprise UI/UX Dashboard

A high-fidelity Figma design for the NTC Region V internal intranet platform. Focused on creating a clean, accessible, and data-rich interface for government staff managing telecom infrastructure across the Bicol region.

Gallery

Problem

NTC Region V was using an outdated intranet system that made it hard for staff to access regulatory data, file documents, and monitor telecom infrastructure. The legacy system had poor navigation, inconsistent UI, and no mobile support — slowing down daily operations.

Solution

Redesigned the platform from scratch in Figma with a focus on clarity and efficiency. Introduced a collapsible sidebar, role-based dashboards, and a dark-themed visual system that reduced eye strain during long working hours. Data tables and KPI cards were redesigned for faster scanning.

Key Features

  • Dark-themed dashboard with regional data charts and KPI cards
  • Collapsible sidebar navigation with grouped module access
  • File management and document upload interface
  • Personnel records module with tabular data layout
  • Role-based access UI differentiation (Admin, Staff, Viewer)
  • Map-integrated telecom infrastructure monitoring screen

Design Process

01

User Research & Audit

Interviewed NTC staff to understand daily workflows and pain points with the legacy system. Conducted a UI audit of existing screens.

02

Wireframing

Produced low-fidelity wireframes for all major screens — dashboard, records, file management, and settings — to establish layout and flow.

03

Design System Setup

Built a Figma component library: color tokens, typography scale, button variants, input states, card templates, and navigation components.

04

High-Fidelity Mockups

Applied the design system to all screens. Iterated on feedback from supervisors, refining data tables, chart layouts, and modal dialogs.

05

Prototype & Handoff

Linked screens into an interactive Figma prototype and prepared developer handoff with specs, assets, and component notes.
