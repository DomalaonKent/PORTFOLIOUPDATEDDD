Travel App

FigmaMobile AppUI/UXTravel

A mobile travel companion app UI designed to help users discover destinations, plan itineraries, and book experiences — all in one place. Vibrant, image-forward design that inspires wanderlust while keeping trip planning simple.

Gallery

Problem

Travel planning is fragmented across multiple apps — one for flights, another for hotels, another for activities. Users waste time switching between platforms and lose context about their full trip plan.

Solution

Designed an all-in-one travel app that consolidates destination discovery, itinerary building, and booking into a single, visually engaging mobile experience. The app uses rich photography and map integration to keep users inspired and organized.

Key Features

  • Destination discovery feed with curated recommendations
  • Interactive itinerary builder with drag-and-drop day planning
  • Integrated map view for nearby attractions
  • Saved trips and wishlist for future travel
  • Activity and experience booking cards

Design Process

01

Mood & Style

Set a vibrant, aspirational visual direction — full-bleed destination photos, warm colors, and clean sans-serif typography to evoke excitement.

02

User Flow

Mapped the core flows: explore destinations → save to trip → build itinerary → book. Kept navigation minimal with a bottom tab bar.

03

UI Design & Prototype

Designed all screens in Figma and connected them into an interactive prototype showcasing the full explore-to-book journey.
